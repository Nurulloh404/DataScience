# Loyihani tekshirish bo'yicha qo'llanma

Bu faylda har bir mashqni (ex00 dan ex09 gacha) qanday ishga tushirish va qanday natija kutish kerakligi yozilgan.
Barcha buyruqlar `src` papkasi ichida turib bajarilishi tavsiya etiladi.

---

## ex00: Data Types
**Fayl:** `ex00/data_types.py`
**Vazifa:** O'zgaruvchilarning turlarini aniqlash va chiqarish.
**Ishga tushirish:**
```bash
python3 ex00/data_types.py
```
**Kutilayotgan natija:**
```
['int', 'str', 'float', 'bool', 'list', 'dict', 'tuple', 'set']
```

---

## ex01: Read and Write
**Fayl:** `ex01/read_and_write.py`
**Vazifa:** `ds.csv` faylini o'qib, ma'lumotlarni o'zgartirib `ds.tsv` fayliga yozish.
**Ishga tushirish:**

**Kutilayotgan natija:**
Ekranda hech narsa chiqmaydi. `src/ex01` papkasida `ds.tsv` fayli paydo bo'ladi (yoki yangilanadi).

## ex08: Names Extractor & Letter Starter
**1-qadam: Names Extractor**
**Fayl:** `ex08/names_extractor.py`
**Vazifa:** Email ro'yxati bor fayldan ism va familiyalarni ajratib `employees.tsv` ga yozish.
**Ishga tushirish:**
Avval ichida emaillar bor fayl kerak (masalan `emails.txt`).
```bash
python3 ex08/names_extractor.py emails.txt
```
*Natija:* `employees.tsv` fayli yaratiladi.

**2-qadam: Letter Starter**
**Fayl:** `ex08/letter_starter.py`

**Ishga tushirish:**
```bash

```
**Kutilayotgan natija:**

## ex09: Caesar Cipher
**Fayl:** `ex09/caesar.py`
**Vazifa:** Matnni Sezar usulida shifrlash (encode) va deshifrlash (decode).
**Ishga tushirish (Encode):**
```bash
python3 ex09/caesar.py encode "Python is cool" 3
```
**Natija:** `Sbwkrq lv frro`

**Ishga tushirish (Decode):**
```bash
python3 ex09/caesar.py decode "Sbwkrq lv frro" 3
```
**Natija:** `Python is cool`
